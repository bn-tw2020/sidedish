## 🌟 화면 별 기능

### 로그인화면

- OAuth2를 활용한 Google 로그인

### 메뉴 화면

- 메인, 국물, 사이드 메뉴 목록
- 메인, 국물, 사이드 메뉴의 총 개수 확인
- 메뉴 선택 가능

### 상세 메뉴 화면

- 메뉴 주문 수량 증가 및 감소
- 메뉴 주문하기
- 메뉴 주문 성공, 실패에 따른 메시지
